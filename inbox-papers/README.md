# inbox-papers/

Drop PDFs or arXiv IDs here for related-work analysis.

For each paper, I will:
1. Read the abstract and contribution framing
2. Identify relationship to LitDiscover (contrast, comparison, complementary, or cited-by)
3. Recommend §2 placement and draft a sentence for the Related Work section
4. Add to bibliography.json if it should be cited

## How to use

- Drop PDF files directly into this folder, OR
- Create a `.txt` file with a list of arXiv IDs / DOIs / paper titles

I will process them at the start of the next session or when you say "check inbox".
